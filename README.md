# AI Music Video Engine

Terminal-first, ComfyUI-based music-video shot generation engine for Windows.

Current repository milestone: **Stage 2 — Stable ComfyUI Startup (v0.2.2)**.

## Target machine

- NVIDIA RTX 4070 Laptop GPU
- 8 GB VRAM
- 16 GB system RAM
- Windows
- One heavy GPU job at a time

## Scope

The engine prepares identities, scene definitions, generation jobs and raw 16:9 clips. Final editing and timeline work remain manual.

## Privacy

Real-person reference photos, identity embeddings, generated videos, models, logs and the local ComfyUI runtime are intentionally excluded from Git.
